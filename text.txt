Git専科

リポジトリの作成
GitBash->cmd : cd <ファイルパス>
GitBash->cmd : git init


GirBash->cmd : git status
赤:変更中
緑:保存済み
何もなし:コミット済み

git --amend 
iキーで入力モード
escキーで離れる
:wqで保存して終了
:qで保存せず終了


GitBash cmd 
git init											//gitの初期化
git status											//ファイルの状態確認
git log												//コミットのログを表示
git add <File Name>									//1ファイルの変更保存
git add .											//全てのファイルの変更保存
git commit -m "<説明文(日本語可能)>"				//プロジェクトの変更保存
git commit --amend									//コミットの編集
git reset HEAD										//addの取り消し
git reset --soft HEAD^								//直前のコミットを削除
git reset --hard <commit:ID>						//IDのコミットに移動
git diff											//変更内容の確認
git push origin master								//リモートリポジトリにGO
